**Fish** is a custom programming language compiled using **Rust**. This project was created as a **learning exercise** for students at the Royal University of Phnom Penh (RUPP) to explore how programming languages and compilers work from the ground up. Fish made by Panha and Jule we're a student at **Royal University of Phnom Penh (RUPP)** Phnom Penh, Cambodia 🇰🇭 as a hobby project

## 🐟 Example Fish Code
 
```fish
# Hello World in Fish!
swim print("Hello, World! 🐟")!
 
# Variables
fin name = "RUPP Student"!
fin age = 20!
 
# Conditionals
if age > 18 {
    swim print("You are an adult fish!")!
} else {
    swim print("Still a baby fish!")!
}
 
# Loop
bubble i from 1 to 5 {
    swim print(i)!
}
```
